核心专利技术

电流模音频放大技术

Current Mode Amplification

全球PCT专利号：US 9,614,483 B2


电流模放大是电流模式音频放大器的简称，是旷世区别于传统放大器的专利音频技术。作为音响系统“发动机”的放大器，基于全分立、特殊拓扑结构的电流模放大器可以做到近似录音现场的聆听体验，让听众真正被打动。同时，在电流模技术的支持下，即使用户使用的是已有的耳机或音箱，并不是专属配合的设计，也会得到比一般设备好得多的效果。

电流模音频放大器使用三极管作为放大器件，全分立架构。它的输入和输出为电压模式，可以与传统电路接口。核心放大部分在电流域处理，恒定为纯A类工作状态。它的放大电路结构与传统电压模放大器(差分输入级，共射电压放大级等)结构完全不同，影响速度和带宽的晶体管级间电容工作在阻抗很低的节点上，不但彻底消除瞬态互调失真（TIMD），而且可以轻松获得更高带宽（满功率状态下500kHz带宽），更低失真的高性能放大（实现0.0002%的失真技术性能）。可以用于任何功率等级（10mW-2000W)和任意尺寸(数平方厘米-专业用途放大器尺寸）。

电流模放大如何工作？

主要由电压控制电流源VCCS，电流模放大单元A(i)，I/V变换级，输出缓冲器（OPT）和负反馈几部分组成。在电子学中，电压和电流都是同时存在的。只是在电流模放大部分，为了获得高性能，信号被电流主宰了而已，输入和输出都是传统的电压型的，这样的设计可以方便兼容更多的讯源和负载。考虑到High-end市场用户强烈的个性，这样的设计可以有更高的系统开放度。

由图示模型可见，输入信号首先进入VCCS（电压控制电流源）。VCCS将输入电压变为电流，送入电流增益级Ai进行电流模放大，再由I/V变换器变为电压后通过输出级（OPT）驱动负载。


整个系统为了保证系统的稳定性，引入了负反馈。由于整个负反馈网络的阻抗只有几百欧姆，加上放大器的高速性能，反馈环路处理速度大约是传统电压模放大器的100倍，所以常规负反馈放大器带来的TIMD等一系列问题都不存在。


纯A类偏压控制

Pure Class A BIAS Control

为了保证电流模放大器以高性能模式达到专业的聆听体验，旷世科技的纯A类偏压控制系统保证了电流模放大器在不同负载的状况下都以严格纯A类方式工作。

旷世科技纯A类偏压控制系统是在旷世专利级电流模放大器基础上，根据用户的设定，通过完全隔离的电路，使用处理器控制并调节和检测电流模放大器输出级的偏压，确保输出级工作在纯A类状态，匹配耳机实现美妙的系统体验。比传统的A类，自动偏压A类等等技术更直接彻底、无负作用，也没有讯号延迟的问题。它能驾驭不同规格耳机或扬声器，轻松释放全部实力。



QP2R是全球使用偏压控制系统的便携音频设备之一。在QP2R的机身侧面，标准偏压时有橙色灯光从微孔中射出。设定为高偏置时，变为红色灯光，输出偏压加大，输出静态电流翻倍，THD+N和输出阻抗降到更低：犹如玩超跑的时候将汽车设定为“RACE模式（竞技模式, 或称Sport Plus或Sport Response 模式）”，在发动机高转速，急速换挡配合强大抓地力，给你直抵地心的征服感无法言喻的满足感。

5GHz无线音频系统

5GHz High-end Wireless Audio System

如果要将高性能的音频体验延伸至家庭的每一个角落或搭载到汽车上，就需要使用区别以往的无线传输技术。High-end 5GHz无线音响系统，源于Questyle。它是一套罕见的兼顾稳定性和高音质的多声道无损无线音频系统。

美国权威影音 ”Home Theater” 将唯一的五星级评价的无线音响系统授予旷世5GHz无线音响系统。

2017年美国洛杉矶音响展上，旷世科技的5GHz无线音响系统，被组委会评选为 “Best Sound” of show awards（最佳音质奖项）。




如果希望将美妙的聆听体验延伸至家庭或者汽车中，使用通用的蓝牙技术传输高清音频不但距离太短，而且有压缩会损失音质；使用WiFi技术又太容易受环境影响，造成卡断和不稳定。另外这些无线技术也有同步困难的问题。工程师不得不把立体声和多声道的终端扬声器设计在一起，但是这样无法还原现场的空间感和方位感，气氛大打折扣。


旷世科技不希望美妙的体验被束缚，针对高性能音频开发了与众不同的5GHz的无线系统。这个独特的技术工作在比较高的RF频率，使用独立的协议栈，不依赖路由器，从本质上解决了传统技术容易受到干扰，不稳定的问题。同时，高码流可以轻松传输无损音频，还支持让人惊讶的8声道同步传输，可以把不同通道的扬声器终端放在合理的位置，并通过了欧盟CE及美国FCC电磁兼容认证，实现了理想的音频无线连接延伸体验。


IIR数字滤波数字信号处理技术

Digital Signal Processing: IIR Digital Filter

中国国家专利局专利号：201520212711.5

为了呈现真实的泛音结构，旷世科技的IIR数字滤波信号处理技术可以彻底消除脉冲波的预振铃。

世界上大多数数字音频系统都是使用FIR（有限脉冲响应）数字滤波器。使用FIR数字滤波器产生的脉冲波预振铃不但非常容易被测试到，而且还很容易被人耳听到，它破坏了音乐本身的连贯和自然。旷世科技先进的IIR数字滤波器数字信号处理技术，完全消除了脉冲波的预振铃，呈现自然的空气流动，接近真实的泛音结构。

True DSD解码技术

True Direct Stream Digital

中国国家专利局专利号：201520211941.X

为了原汁原味地还原DSD音频数据的体验，旷世科技的真DSD（True DSD）专利技术在处理DSD信号时不经过任何PCM转码，保证了彻头彻尾的DSD处理，它和传统的DoP，Native DSD有本质不同。

当系统处理DSD数据时，系统硬件模式立即改变，切换到True DSD硬件通道处理数据。在这种情况下，所有的数字滤波器选择失效，所有数字音量控制失效，严格遵循SACD回放的流程处理。

3x时钟异步架构

3x Clock USB Asynchronous Transmission

中国国家专利局专利号：201520211445.4

为了保证在无损数字音频系统回放时的高性能，避免数据信号的时基抖动，旷世科技的3x时钟异步架构使用3时钟结合异步FIFO数字逻辑硬件控制和算法，避免传统做法引起的时基误差（Jitter)，保证了数字音频文件在回放的时候的基本素质。

3x时钟异步架构，第一枚时钟负责系统，第二枚时钟来驱动44.1K及44.1K倍数的采样信号， 第三枚时钟用来驱动48K及48K倍数的采样信号和DSD信号。

双时钟整倍升频设计

Dual Clock Integral Upsampling

中国国家专利局专利编号: ZL201520211445.4

为了克服传统升频电路的诸多不足，双时钟整倍升频设计用2个低相位噪声时钟源，自动判断当前采样率，整倍升频，消除引入误差。

大多数升频设计，不管信号采样率是多少，全部升频到192K处理。而非整倍数重采样转换时，数据运算会引入误差和失真，劣化音质。双时钟整倍升频则与之不同，内部有2个低相位噪声时钟源，处理器自动判断当前采样率，并按照整倍数选择升频，消除属于引入误差，提高音质。
